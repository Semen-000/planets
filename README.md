# 🚀 Мега-симуляция Солнечной системы 3000

Космический эпос на Python с использованием Tkinter и Pillow. Интерактивная 2D-симуляция Солнечной системы с красивыми визуальными эффектами, реалистичными текстурами и множеством функций.

## 🌟 Возможности

- **8 планет + Плутон и Церера** с реалистичными параметрами
- **Реальные фотографии планет** (Меркурий, Венера, Земля, Марс, Юпитер, Сатурн, Уран, Нептун)
- **Пользовательское изображение Солнца** с эффектом свечения
- **Естественные спутники**: Луна, Фобос, Деймос, спутники Юпитера, Сатурна и других планет
- **Пояс астероидов** между Марсом и Юпитером
- **Туманности и объекты глубокого космоса**
- **Кометы** со следами
- **Черные дыры** с аккреционными дисками
- **Интерактивное управление**:
  - Выбор планет кликом мыши
  - Масштабирование колесиком (с плавной анимацией)
  - Перетаскивание для перемещения по карте
  - Двойной клик для центрирования на планете
  - Горячие клавиши (пробел, C, R, F, A, +, -, H)
- **Режимы отображения**: орбиты, названия, сетка, эффекты, миникарта, легенда
- **Детальная информация** о каждой планете при клике
- **Автоматический зум** для обзора всей системы
- **Режим слежения** за выбранной планетой
- **Скриншоты** и сохранение/загрузка симуляции

## 🎮 Управление

| Действие | Описание |
|----------|----------|
| 🖱️ ЛКМ | Выбрать планету |
| 🖱️ Перетаскивание | Переместить вид |
| 🖱️ Колесико | Изменить масштаб |
| Ctrl + 🖱️ Колесико | Точная настройка зума |
| Shift + 🖱️ Колесико | Горизонтальный скролл |
| 🖱️ ПКМ | Контекстное меню |
| 🖱️ Двойной клик | Центрировать на планете |
| Пробел | Пауза |
| C | Центрировать вид |
| R | Сброс симуляции |
| F | Следить за планетой |
| A | Автоматический зум |
| + / - | Быстрый зум |
| H | Скрыть/показать панель управления |

## 🔧 Установка

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/Semen-000/planets.git
cd planets
```
2. **Установите зависимости:**

```bash
pip install Pillow
```
3. **Добавьте изображения планет в корневую папку:**

🎯 Особенности реализации

- Анимированное звездное поле с параллаксом

- Реалистичные орбиты с разными скоростями вращения

- Правильное масштабирование изображений с сохранением пропорций

- Атмосфера планет (Венера, Земля, Марс, Юпитер, Сатурн, Уран, Нептун)

- Кольца Сатурна с детальной прорисовкой

- Большое Красное Пятно на Юпитере

- Солнечные пятна и корона

- Спутники планет (Луна, Фобос, Деймос, Ио, Европа, Ганимед, Каллисто, Титан, Титания, Оберон, Тритон, Харон)

- Информационная панель с подробными данными о каждой планете

📊 Данные о планетах

- Для каждой планеты доступна информация:

- Физические характеристики (радиус, масса, плотность, сила тяжести, температура)

- Орбитальные характеристики (расстояние от Солнца, период, скорость)

- Состав (атмосфера, кольца, спутники)

- История открытия

- Интересные факты
