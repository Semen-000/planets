# Нейрободр - Система идентификации операторов

## Что это?
Десктопное приложение для регистрации и авторизации операторов с биометрической привязкой. Разработано в рамках задания neuro-awake_Task006.

### Функционал:
- 👤 Регистрация новых операторов с сохранением в CSV
- 🔐 Авторизация по ID оператора
- 📸 Загрузка фотографий с проверкой размера (800x600 px)
- ✅ Верификация оператора
- 📊 Отображение информации об операторе

## Как работает?

### Основные окна:

**Окно №1 - Стартовая форма**
Выбор режима работы: регистрация или авторизация.

**Форма регистрации**
- Ввод ФИО и возраста
- Сохранение в CSV с автоматическим присвоением ID
- Загрузка фото оператора

**Авторизация**
- Ввод ID оператора
- Отображение информации об операторе

**Верификация**
- Проверка фото на соответствие размеру
- Отображение результата проверки

### Структура данных:
- `operation_db.csv` - база данных операторов
- `operations/` - папка с фотографиями операторов (ID_номер.jpg)
